# iMessage Relationship Stats

Beautiful visualization of your top iMessage contacts with detailed relationship statistics.

## Features

- 📊 Top 25 contacts by message volume (1-on-1 only)
- 💬 Message and session counts
- ⚡ Response time metrics
- 🔥 Conversation streaks
- 📈 Message ratio visualizations
- 🎯 Initiator ratio tracking

## Data Privacy

All data is generated locally from your iMessage database. No data is sent to any server.
